# ScizersTechnologiesLLP

 ## Installation
1. Clone the repository:
    ```sh
     https://github.com/prathameshpanse51/zoroz-ecommerce.git
    ```
2. Install dependencies:
    ```sh
      npm install
    ```
3. Run the project locally to see all features in action:
    ```sh
      npm run dev
    ```
